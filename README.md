# 🤖 Friday AI Assistant

Friday AI Assistant is a Python-based desktop voice assistant that recognizes voice commands and performs everyday tasks such as opening websites, launching applications, and responding using text-to-speech.

## ✨ Features
- 🎤 Voice command recognition
- 🔊 Text-to-speech responses
- 🌐 Opens Google, YouTube, Gmail, Spotify, ChatGPT
- 💻 Launches desktop applications
- ⚡ Fast and easy to use

## 🛠️ Tech Stack
- Python
- SpeechRecognition
- pyttsx3

## 📂 Project Structure
- main.py
- voice.py
